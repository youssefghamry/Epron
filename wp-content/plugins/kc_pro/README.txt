== Description ==

- A premium front-end live editor addon for KingComposer page builder with super fast, flexible and real-time preview.


== Installation ==

IMPORTANT: This plugin only works when the KingComposer plugin is installed. You can download it here https://wordpress.org/plugins/kingcomposer/

e.g.

= Install directly via KingComposer dashboard =

1. Install the KingComposer plugin by search the keywords "kingcomposer" in your plugins 
2. Navigate to the 'Upload' area
3. Select `kc_pro` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `kc_pro` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `kc_pro.zip`
2. Extract the `kc_pro` directory to your computer
3. Upload the `kc_pro` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
